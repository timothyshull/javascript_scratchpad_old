Enjoy Your App!

Finally you can tie everything together and enjoy your app. This is done by registering, starting and stopping your modules. When you register modules you can attach them to hubs other than 'main'. When starting modules you can pass in configuration that gets applied to the module. Check out the api doc for details. 


Kernel.register('module-box', 'StatusBox');
Kernel.register('module-count', 'StatusCount');
Kernel.register('module-list', 'StatusList');

$('document').ready(function() {
    
    Kernel.start('module-box', {renderTo: '#top'});
    Kernel.start('module-count', {renderTo: '#center'});
    Kernel.start('module-list', {renderTo: '#bottom', title: 'My Title'});
    
});
Using multiple hubs ('main' and 'legacyHub') 


Kernel.register([
    {id: 'module-box', type: 'StatusBox', config: {renderTo: '#top'}},
    {id: 'module-count', type: 'StatusCount', config: {renderTo: '#center'}},
    {id: 'module-list-legacy', type: 'StatusList', hub: 'legacyHub', config: {renderTo: '#bottom', title: 'My Title'}},
]);

$('document').ready(function() {
    
    Kernel.startAll();
    
});
